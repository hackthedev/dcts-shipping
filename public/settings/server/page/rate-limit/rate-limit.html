Rate Limit

This is the new and advanced rate limit system! It will automatically adjust itself based on community activity and will be as accurate as to the hour!

To disable the rate limit or slow mode, set the value to 0 The record history setting supports the following formatting:
  • -/+ second(s)
  • -/+ minute(s)
  • -/+ hour(s)
  • -/+ day(s)
  • -/+ month(s)
  • -/+ years(s)
Valid Examples:
  • 7 days
  • 1 month
  • ...
